## Stack

- astro
- typescript
- react
- tailwind
- react-icons
- shadcn/ui
- prettier
- locomotive-scroll

## Contributing

1. [Fork](https://github.com/slydragonn/alogocode.site/fork) the repository.

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

Open up [http://localhost:4321/](http://localhost:4321/)
